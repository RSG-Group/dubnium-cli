# dubnium-cli

The boilerplate manager to rule 'em all. Language agnostic, built upon Git and highly configurable. Unifies angular-cli, vue-cli, CNA, CRNA and CRA.

## Install

```zsh
npm install --global dubnium-cli
```

## Usage

```zsh
$ dubnium --help

  Usage
    dubnium [input]

  Options
    --foo  Lorem ipsum [Default: false]

  Examples
    $ dubnium
    unicorns & rainbows
    $ dubnium ponies
    ponies & rainbows
```
